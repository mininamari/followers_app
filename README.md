# Novakid Social Reports

Командное веб-приложение для расчета подписчиков из Instagram/Meta и Novakid PR CSV.

## Что внутри

- красивый UI в стиле Novakid: яркий hero-блок, карточки, боковое меню, метрики и графики;
- логин пользователей;
- роли `admin` и `user`;
- разделение по аккаунтам/регионам: `novakid_israel`, `novakid_france` и т.д.;
- отдельная загрузка Meta Business Suite;
- отдельная загрузка Novakid PR таргетологом;
- автоматический матчинг по `account + period + ID публикации`;
- ручное уточнение подписчиков из рекламного кабинета для выбранных строк;
- финальный отчет со ссылками на публикации;
- графики по месяцам и аккаунтам;
- общие фильтры региона и периода для Dashboard и Reports;
- хранение только данных аккаунтов Novakid, без аккаунтов блогеров;
- история загрузок;
- экспорт отчета в CSV и Excel.

## Первый запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Первый вход:

```text
login: admin
password: admin123
```

После первого входа создайте новых пользователей и смените пароль администратора.

## Как работает расчет

В финальный отчет попадают все публикации из Meta Business Suite, где подписчиков 1 или больше.

Join:

```text
Meta Business Suite: ID публикации
Novakid PR: Название объявления
```

Финальные подписчики:

```text
Подписчики Meta - Подписчики Novakid PR для расчета
```

По умолчанию для расчета используются подписчики Novakid PR из CSV. В разделе
`Reports` можно выбрать отдельные строки и указать фактическое значение вручную.
Ручное значение сохраняется отдельно, имеет приоритет над CSV и не пропадает при
повторной загрузке файлов. Очистка ручного значения возвращает расчет к CSV.

Если значение отрицательное, система ставит 0 и добавляет предупреждение.

CPF:

```text
Spend / Подписчики Novakid PR для расчета
```

## Для сервера

Текущая версия использует SQLite для простого локального запуска. Для командной production-версии лучше заменить SQLite на PostgreSQL и развернуть приложение на Render, Railway, DigitalOcean или другом сервере с HTTPS.
